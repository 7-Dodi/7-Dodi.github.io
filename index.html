Um pouco sobre mim

Olá meu nome é Douglas. Sou estudante de programação, estou no segundo período do curso de Análise e Desenvolvimento de Sistemas. Faço alguns projetos em Back-End e em Front-End. Sou Apaixonado por tecnologia, educação e por mudar a vida das pessoas através da programação.

Criação de jogos 2D, com Java Script, HTML e CSS:

A criação de jogos é a parte mais divertida do estudo de programação, sempre tiver vontade de desenvolver jogos, mesmos que simples, e atualmente os jogos 2D têm me chamado a atenção. O projeto ao lado é o jogo mais recente desenvolvido por mim, consiste em um jogo em que o personagem deve pular por cima dos vilões, parecido com o jogo do dinossauro do Google.

Desenvolvimento de sites e/ou E-commerce:

O desenvolvimento de sites pessoais, para empresas e/ou trabalho, é a principal forma que uso para o meu desenvolvimento no mundo da programação Front-End. Este projeto é um dos mais recentes, foi desenvolvido com intuito de atender a necessidade de um ateliê de artesanato, facilitando a divulgação dos produtos e a comunicação entre produtor e cliente.

Desenvolvimento de Dashboard com Linguagem R:

A linguagem é uma das minhas recentes descorbertas no mundo da programação; consiste em uma linguagem de análise e desenvolvimento de gráficos e cálculos probabilísticos. Por essa finalidade tenho buscado me desenvolver nessa área de pesquisa. A criação e/ou desenvolvimento de dashboard contribuem para desenvolvimento de pesquisas e para o desenvolvimento de softwares e apps. Tenho um exemplo simples desses dashboard publicados(a critério de estudo), pode acessar clicando no botão "Ver site".

Desenvolvimento de códigos e/ou algoritmos em C:

Atualmente essa é linguagem que mais uso para o desenvolvimento de softwares e sistemas de dados. Essa foi a primeira lingaugem de programação que aprendir e tenho um grande sobre. É uma linguagem que não é muito usada, mas altualmente, é a linguagem que tenho um maior domínio; estou a estudar outras linguagens como: Lua, C++ e Java.
